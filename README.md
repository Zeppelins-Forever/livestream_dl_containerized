# livestream_dl_containerized
A docker container and scripts for using livestream_dl conveniently!


Run with:
` sudo docker run -v "$(pwd):/out" zeppelinsforever/livestream_dl_containerized:latest [URL] ` or ` docker run -v "$(pwd):/out" zeppelinsforever/livestream_dl_containerized:latest [URL] ` depending on if you have a docker user set up

A Script to make this process easier will be available soon-ish.
